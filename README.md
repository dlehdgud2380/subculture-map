## Tech Stack
### Backend
* Python 3.9 -> 개발언어
* Django 4.0.8 -> 웹프레임워크
* DRF 3.13.1 -> Restful Server 구축을 위한 프레임워크
* Pillow 9.3.0 -> 이미지 처리
### Frontend
* Flutter -> iOS, Android
* HTML5 with BootStrap 5 -> Web
* Axios -> HTTP통신
